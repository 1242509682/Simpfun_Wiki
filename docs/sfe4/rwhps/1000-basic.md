
---
title: 了解运行环境
---


第一次运行会初始化运行环境。下表说明了各个文件夹的用途。


|             文件夹名称             | 用途                               |
|:----------------------------------:|:-----------------------------------|
|        `data/plugins`         | 存放插件                                |
|          `data/maps`          | 存放地图                                |
|          `data/mods`          | 存放Rwmod                               |
|         `data/cache`          | 存放缓存，一般不需要在意它们            |
|          `data/log`           | 存放Log，一般不需要在意它们             |
|      `data/Config.json`       | 存放服务器配置，可以打开并修改配置      |
|      `data/Settings.bin`      | 存放内部配置，一般不需要在意它们        |
